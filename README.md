# Age Calculator

A simple web-based Age Calculator built with HTML, CSS, and JavaScript.

## Overview

This project calculates a user's age in years, months, and days based on the selected birth date.

## Features

- Date input field with a maximum selectable date of today
- Calculates age in years, months, and days
- Responsive and colorful design
- Pure client-side implementation using JavaScript

## Files

- `index.html` - Main page with date picker and result output
- `style.css` - Styling for layout and appearance

## How to Use

1. Open `index.html` in your web browser.
2. Select a birth date from the date picker.
3. Click the "Calculate" button.
4. The app will display your age in years, months, and days.

## Notes

- The age is calculated relative to the current date.
- The date picker prevents selecting future dates.
